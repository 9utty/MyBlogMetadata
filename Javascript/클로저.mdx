## 클로저가 뭘까?(면접에서 단골질문 중 하나)

- 자바스크립트에서 클로저는 함수와 그 함수가 생성된 렉시컬 선언 환경(Lexical Environment)으로 이루어진 특별한 개념이다. 즉, 함수 내부에서 정의된 내부 함수가 자신의 외부 함수의 변수나 매개변수를 기억하고 접근할 수 있는 것을 말한다.

## 클로저는 어떤 상황에서 써야할까?

- 데이터 은폐 및 캡슐화
- 메모이제이션 (계산 결과를 저장하고 재사용)
- 함수 간에 상태를 공유하려는 경우

## 예제 코드

```typescript
function makeCounter() {
  let count = 0;

  return function () {
    return count++; // 여기서 클로저가 발생
  };
}

const counter = makeCounter();
console.log(counter()); // 0
console.log(counter()); // 1
console.log(counter()); // 2
```

- 위 코드에서 makeCounter 함수는 내부 함수를 반환한다. 이때 내부 함수는 외부 함수인 makeCounter의 count 변수에 계속해서 접근이 가능하며, 이것이 클로저의 특징이다.

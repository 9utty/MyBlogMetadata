1. NextJS를 이용한 정정페이지를 만들기 시작했습니다

- NextJS로 정적페이지를 추출할 수 있어요! "next build && next export" cmd 명령어로 빌드하면 된다고 합니다!

2. 배포는 GithubPage? Netlify?

- 제가 선택한건 Netlify라는 정적 웹 호스팅을 해주는 배포서비스를 이용했습니다!
  - 그 이유로는 생각보다 초보가 사용하기 쉽게 잘 되어 있고, 배포도 간단하게 Netlify에서 github Repo와 연겷만 잘 해놓으면 자동으로 되기 때문에 선택했습니다

3. 데이터들은 어떻게?

```typescript
interface Menu {
  key: string;
  value: string;
}
```

- 프로젝트 내에서 그냥 Menu[]를 만들어두고 그 안에 데이터들을 정적으로 생성하고 사용했습니다!
- key는 음식의 카테고리(예: 중식, 한식, 양식), value는 음식 이름(예: 탕수육, 짜장면, 김치찌개)
